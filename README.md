## 安装步骤
- 执行 install-dependence.sh 安装依赖
- 编译安装 st-0.8.4
- 编译安装 dwm
- 重命名scripts为 ~/.dwm
- 复制dwm.desktop文件到/usr/share/xsessions 目录里
